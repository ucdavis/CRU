## TA imports from Banner

Ace now imports Teaching Assistants from Banner when an evaluation is created. This import happens only once for each course in the current term.

Because the import is a one-time process, the instructor roster is not replaced or refreshed if changes are made in Banner after the evaluation is created. Make sure Banner has the correct TAs populated before creating evaluations so they can be imported along with the instructors.

After evaluations are created, review them to confirm the instructor and TA roster is correct.

> **Note:** Deleting an evaluation and creating it again does not re-import the roster from Banner.
